JChemPaint User's Guide
July 2005

This user's guide contains the following sections:

About JChemPaint

General information about the JChemPaint program and the JChemPaint project.

Tutorial

A guide for help how to: new compounds and reactions can be drawn.

Reference Guide

A general overview of the JChemPaint. Describes the use of JChemPaint and how it can be customized. It also tried to describe as best as possible which algorithms are used, and in which CDK classes those can be found.

Beyond JChemPaint

This chapter describes how JChemPaint can be extended to provide even more functionality, or fit it into your corporate working environment with specialized plugins. Because JChemPaint is only where it starts

Localizing Help Information

A guide for localizers JChemPaint.
