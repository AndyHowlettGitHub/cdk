If you want to get the result more rapidly, please do not input big

molecular formula, normally the number of non-hydrogen atoms

should be no more than 10 for unsaturated compounds. For saturated

molecular formula, it should be 15 or so. Even that, sometimes, some

generation step should be also very slow, such as C9.
