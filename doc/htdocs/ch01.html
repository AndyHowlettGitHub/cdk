What is the CDK?

The CDK classes are Java utitility classes for ChemoInformatics and Computational chemistry, written in Java. They are developed constantly developed parallel to other projects that make use of them. They are a complete re-write of the CompChem classes that were the basis of JChemPaint, a Java Editor for 2D chemical structures, and of JMDraw, a Java package for the graphical layout of 2D chemical structures

A re-write was needed to in order to assure the widest possible usability. It is also intended to port JMol, a 3D molecular visualization tool, to the CDK.

The CDK classes support (or will support, as soon as ported from the old CompChem library):

  • Classes for Atoms, Molecules and other chemical concepts.

  • Utilities for importing and exporting chemical structures (Molecules or arrays of Atoms) using standard chemical files format.

  • Utility classes for handling chemical elements and isotopes.

  • A class that parses a molecular formula (a String of characters) and returns an array of Nodes.

  • An isomorphisms tester that compares to structures and returns if they are isomorph. I'm almost sure, that the implementation is not strictly valid, but it seems to work

  • A SMILES parser (hidden as a static method in compchem.ChemFileUtils), which so far only implements the SSMILES standard.

  • An implementation of John Figueras, "Ring Perception Using Breadth-First Search", J. Chem. Inf. Comput Sci. 1996, 36, 986-991.

Please check the existing API documentation for information on which classes are already implemented.
