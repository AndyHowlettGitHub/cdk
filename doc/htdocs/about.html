About the CDK Homepages

These web pages are dedicated to the documentation, developement and discussions on the CDK, an API for writing java program for chemistry. They have been produced based on a Docbook XML source and have been converted to this website using Norman Walsh's XSLT website stylesheets. We are also producing a PDF manual using the same XML source and the Norm's docbook XSLT transformation sheets
