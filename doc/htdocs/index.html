News

  • (12. Dec. 2000) On SourceForge a small survey (14 questions) is available to get an idea on the usage and usability of JChemPaint.

  • (11. Dec. 2000) JChemPaint has now the ability to print Postscript files to both a printer and to file.

  • (6. Dec. 2000) The CVS version now includes a working version of the Java SVG Toolkit, called Batik. A screenshot of JChemPaint and the Batik SVG viewer as available. Scalable Vector Graphics (SVG) is a new W3 Consortium standard, and is supported by Mozilla.

  • (4. Dec. 2000) The Applet on http://jchempaint.sourceforge.net/jcpapplet has been updated to contain all the features of JCP 0.7 (SMILES import, ring double bonds nicely drawn, etc.). Martin Schild has also added a demo of how to use JavaScript for the communication with the applet.

  • (25. Nov. 2000) There is now an applet version, which can be found on http://jchempaint.sourceforge.net/jcpapplet

  • (1. Nov. 2000) JChemPaint was mentioned in an article on Chemical Visualisation in "Nachrichten aus der Chemie", which is the members journal of the German Chemical Society written by Frank Oellien, Thomas Engel and Markus C. Hemmer from the Computer Chemistry Center in Erlangen, Germany

  • (21. Aug. 2000) JChemPaint has been mini-reviewed in the Software Briefs in Chemical & Engineering News, the members journal of the American Chemical Society.
